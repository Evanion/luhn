Static validate

  • validate(input: string, sensitive?: boolean): { isValid: boolean; phrase: string }
    • Defined in src/lib/luhn.ts:78

    Validates if the given string, matches the calculated checksum. The last character in the string is considered the checksum.

    example

    Luhn.validate('foo5') // -> {phrase: 'foo5', isValid: true} Luhn.validate('FoO5') // -> {phrase: 'foo5', isValid: true} Luhn.validate('FoOö5') // -> {phrase: 'foo5', isValid: true} Luhn.validate('FoO5', true) // -> {phrase: 'FoO5', isValid: false}

    Parameters

    • input: string

      The string that you want to check, including the check character in the end of the string

      Toggle if the function should be case-sensitive or not.
